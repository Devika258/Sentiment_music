Sentiment-Based Music Recommender

System Architecture

The application follows a modular, cloud-native architecture:

  • User Frontend: Simple interface to submit moods and view playlists
  • FastAPI Backend: Manages authentication, mood analysis, API calls
  • External APIs: OpenAI (Sentiment analysis) and Spotify (Playlist suggestions)
  • PostgreSQL Database: User accounts, credits, history, saved playlists

Deployment Strategy

  • FastAPI backend on AWS Lambda or ECS (Dockerized)
  • Database hosted via AWS RDS (PostgreSQL)
  • Frontend deployed on Netlify or Vercel
  • External API keys (OpenAI, Spotify) managed securely via AWS Secrets Manager
